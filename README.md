# Actividad – Programación 3

## ¿Para qué sirve el archivo `package.json`?

El archivo pacjage.json sirve para: definir el nombre, versión y descripción del proyecto, registrar todas las dependencias y devDependencies que necesita la aplicación, guardar los scripts que permiten ejecutar comandos como npm run dev, npm run build, etc. Mantener un control claro de las librerías instaladas para que cualquier persona pueda instalar el proyecto con solo ejecutar npm install.

---

## ¿Qué diferencia notaron entre ejecutar un archivo directamente con Node.js y levantar el servidor de React?

Ejecutar un archivo con node.js es directo y sencillo y levantar un servidor con react lleva mas pasos y complejidad.
